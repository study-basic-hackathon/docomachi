# Quickstart: 007-frontend-design

**Feature**: Screen Layout and Visual Polish  
**Branch**: `007-frontend-design`

## 前提

- Node 20+
- リポジトリルートで `npm install` 済み
- （Amplify 利用時）`amplify_outputs.json` が設定されていること

## セットアップ

```bash
# リポジトリルートで
git fetch origin
git checkout 007-frontend-design
npm install
```

## 開発サーバー起動

```bash
npm run dev
```

ブラウザで http://localhost:3000 を開く。

## 確認ポイント（本機能）

1. **トップページ**
   - 画面上部にトップバナー画像（`/top-banner.png` または `/images/top-banner.png`）が表示されていること。
   - 背景が緑系（麻雀マット風）で、メインコンテンツが中央寄せであること。
   - 「スタート」ボタンを押すと、オーバーレイのローディングが表示され、その後クイズページに遷移すること。

2. **クイズページ**
   - 待ち牌選択エリアで、牌が **色（萬・索・筒・字牌）ごとに改行** して表示されていること。
   - 牌を選択すると、選択中は **border などで強調** されていること。
   - 牌やボタンが多くて画面に収まらない場合、**その領域内でスクロール** できること。
   - 問題読み込み中は **画面全体またはオーバーレイ** でローディングが表示されること。
   - エラー・タイムアウト時はローディングが消え、**トースト／スナックバー** でメッセージが表示されること。

3. **スマホ表示（狭いビューポート）**
   - 横表示で固定されるか、または縦でもレイアウトが横長で使いやすいこと（Screen Orientation API または CSS による対応）。

4. **タブレット・デスクトップ**
   - 端末の向きに応じてレイアウトが切り替わり、縦横どちらでも利用可能であること。

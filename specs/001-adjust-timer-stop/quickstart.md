# Quickstart: 解答タイマー15秒化と停止

## 目的

制限時間を15秒へ変更し、解答ボタン押下でタイマーを停止する仕様を実装・検証する。

## 前提

- ブランチ: `001-adjust-timer-stop`
- 既存クイズ画面が起動可能
- Jest テスト実行環境が利用可能

## 実装ステップ

1. `app/quiz/page.tsx` の制限時間定数を15秒へ変更する。
2. 「解答する」押下時にタイマー更新を停止する処理を追加する。
3. 結果モーダル表示中に残り時間表示が更新されないことを担保する。
4. 競合ケース（押下と時間切れ同時）で時間切れ優先となるよう統一する。
5. 次問題遷移時にタイマーを15秒で再初期化する。

## 最小テスト観点

- 15秒以内回答は通常判定される。
- 15秒経過未回答は不正解モーダルが表示される。
- 解答ボタン押下後、残り時間表示が停止する。
- 競合ケースで確定は1回のみ行われる。
- 15秒経過と同時押下では時間切れ優先となる。

## コマンド

- `npm test -- --runInBand app/quiz/page.test.tsx`

## 完了条件

- `spec.md` の FR-001 から FR-007 を満たす。
- 仕様変更点（15秒化、押下時停止）が再現テストで確認できる。

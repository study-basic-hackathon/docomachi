# Quickstart: UI Visibility Update

## 1. 前提

- ブランチ: `001-ui-visibility-timer`
- 依存関係インストール済み (`npm install`)

## 2. 開発サーバー起動

```bash
npm run dev
```

## 3. 検証対象

- クイズ回答画面で以下を確認する:
  - 必須テキストが白背景上で判読可能
  - 低視認色（黄緑系など）が必須情報に使われていない
  - タイマーが解答ボタンの直上に表示される
  - タイマー表示サイズが 24 相当未満にならない

## 4. 受け入れチェック手順

1. クイズを開始し、問題表示状態を確認する。
2. 問題文・選択肢・ステータス表示の可読性を確認する。
3. タイマー位置が解答ボタン直上で固定されることを確認する。
4. 小さい画面幅でもタイマーが非表示にならず、解答導線付近に表示されることを確認する。
5. 基準不適合がある場合はスクリーンショットと対象要素を記録する。

## 5. 最低限のテスト更新方針

- 既存Jestテストに対して、表示条件に関する最小ケースを追加する。
- 追加対象の例:
  - タイマー配置コンポーネントの描画位置検証
  - 必須テキストスタイル設定のスナップショットまたは属性検証
